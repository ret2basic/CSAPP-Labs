# CSAPP Labs

Computer Systems: A Programmer's Perspective (CSAPP) is a comprehensive introduction to low-level computer science. The topics covered are:

- A Tour of Computer Systems
- Representing and Manipulating Information
- Machine-Level Representation of Programs
- Processor Architecture
- Optimizing Program Performance
- The Memory Hierarchy
- Linking
- Exceptional Control Flow
- Virtual Memory
- System-Level I/O
- Network Programming
- Concurrent Programming

The essence of CSAPP is the labs. Together with reading the book, we are going to implement the following companion labs:

- Data Lab
- Bomb Lab
- Attack Lab
- Buffer Lab
- Architecture Lab
- Cache Lab
- Performance Lab
- Shell Lab
- Malloc Lab
- Proxy Lab

The material for these labs can be downloaded at http://csapp.cs.cmu.edu/3e/labs.html.
